也可以通过 API 接口上传插件:

curl -X POST {{ request.host_url }}api/packages/publish \
  -F "PluginId=Spectrum" \
  -F "Name=Spectrum Plugin" \
  -F "Version=1.0.0.1" \
  -F "Description=光谱分析插件" \
  -F "Author=xincheng" \
  -F "Category=Analysis" \
  -F "package=@Spectrum-1.0.0.1.cvxp"

或使用 publish_plugin.py 脚本:

python Scripts/publish_plugin.py \
  -p Spectrum -v 1.0.0.1 \
  -f Spectrum-1.0.0.1.cvxp \
  --api-url {{ request.host_url.rstrip('/') }}
上传成功后会刷新插件摘要缓存，列表页可立即看到新版本。
超过保留数的旧包会自动归档到 `History/Plugins/...`。
如果是自动化发布，优先使用 API 以获得更完整的元数据写入。
