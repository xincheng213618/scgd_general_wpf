# 快速上手

本頁面向第一次接觸 ColorVision 的讀者，目標是在最短路徑內完成一次“啟動程式 -> 識別介面 -> 做一個基本操作 -> 知道後面該看哪裡”的閉環。

## 開始之前

建議先完成以下準備：

1. 已確認 [系統要求](./prerequisites.md)
2. 已完成 [安裝指南](./installation.md)
3. 可以正常啟動主程式

## 10 分鐘體驗路線

### 第 1 步：啟動並確認主介面正常

- 啟動 ColorVision
- 確認主視窗可以正常顯示
- 如果首次啟動較慢，先等待初始化和外掛掃描完成

想先熟悉介面結構，可以直接閱讀 [主視窗導覽](../01-user-guide/interface/main-window.md)。

### 第 2 步：認識幾個核心區域

建議先只關注這幾個區域：

- 選單欄：進入檔案、裝置、流程、幫助等主要功能
- 中央工作區：檢視影像、流程或結果
- 屬性面板：檢視和修改當前物件參數
- 狀態列與日誌：確認程式是否報錯、當前狀態是否正常

### 第 3 步：做一次最小操作

如果你只是想確認程式可用，優先選下面兩種最簡單的路徑之一：

#### 路徑 A：開啟一張圖片

1. 從檔案選單開啟本地圖片
2. 確認影像能在工作區顯示
3. 觀察屬性面板或相關工具是否正常響應

#### 路徑 B：新增一個模擬或現有裝置

1. 進入裝置相關選單
2. 新增一個模擬裝置，或連線當前已有裝置
3. 確認裝置能出現在裝置區域並正常響應

裝置相關內容可以繼續看 [裝置服務概覽](../01-user-guide/devices/overview.md)。

### 第 4 步：嘗試一個更接近真實業務的入口

根據你的目標選擇後續閱讀：

- 想做影像檢視和標註：前往 [影像編輯器概覽](../01-user-guide/image-editor/overview.md)
- 想配置裝置和採集鏈路：前往 [裝置服務概覽](../01-user-guide/devices/overview.md)
- 想開始理解流程如何設計和執行：前往 [工作流程概覽](../01-user-guide/workflow/README.md)
