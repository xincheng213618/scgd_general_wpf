## 為什麼這裡不再直接列能力清單

- 舊版特性列表裡混入了難以從當前倉庫直接核對的描述

- 當前結構已經把“安裝與首次使用”“日常使用”“設計與架構”“API 參考”拆開，讀者應沿著這些入口進入
