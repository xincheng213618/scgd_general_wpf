# 首次執行指南

本指南幫助您在首次安裝 ColorVision 後，瞭解如何啟動程式並進行基本配置。

## 啟動程式

### 方式一：從開始選單啟動

1. 開啟 Windows 開始選單
2. 找到 "ColorVision" 或 "視彩光電"
3. 點選圖示啟動程式

### 方式二：從桌面快捷方式啟動

如果安裝時選擇了建立桌面快捷方式，雙擊桌面上的 ColorVision 圖示即可啟動。

### 方式三：從命令列啟動（開發者）

```powershell
cd "安裝目錄"
.\ColorVision.exe
```

## 首次啟動時的自動配置

程式首次啟動時會自動執行以下操作：

1. **建立配置檔案**
   - 位置：`%USERPROFILE%\Documents\ColorVision\`
   - 包含：使用者配置、裝置配置、流程配置等

2. **初始化日誌系統**
   - 日誌目錄：`%USERPROFILE%\Documents\ColorVision\logs\`
   - 日誌級別：預設為 Info

3. **掃描外掛**
   - 自動掃描 `Plugins` 目錄
   - 載入所有可用外掛

4. **顯示主視窗**
   - 展示主介面
   - 顯示歡迎提示（可選）

## 基本介面導覽

### 主視窗組成

```
┌─────────────────────────────────────────────┐
│  選單欄                                     │
├─────────────────────────────────────────────┤
│  工具欄                                     │
├──────────┬──────────────────────┬───────────┤
│          │                      │           │
│  裝置列表 │   影像顯示區域       │  屬性面板  │
│          │                      │           │
│          │                      │           │
├──────────┴──────────────────────┴───────────┤
│  狀態列                                     │
└─────────────────────────────────────────────┘
```

### 主要區域說明

- **選單欄**：檔案、編輯、檢視、裝置、流程、外掛、幫助等選單
- **工具欄**：常用功能快捷按鈕
- **裝置列表**：顯示已連線和可用的裝置
- **影像顯示區域**：顯示影像、流程編輯器等
- **屬性面板**：顯示選中物件的屬性
- **狀態列**：顯示程式狀態和提示資訊

詳細的介面說明請參考：[主視窗導覽](../01-user-guide/interface/main-window.md)

## 基本操作演示

### 1. 開啟影像

**步驟**：
1. 點選選單欄 **檔案** → **開啟影像**
2. 或使用快捷鍵 `Ctrl+O`
3. 選擇影像檔案（支援 BMP, JPG, PNG, TIFF 等格式）
4. 點選"開啟"

影像將在中央顯示區域顯示。

### 2. 新增模擬裝置

**步驟**：
1. 點選選單欄 **裝置** → **新增裝置**
2. 選擇 **模擬相機** 或其他模擬裝置
3. 配置裝置參數
4. 點選"確定"

裝置將出現在左側裝置列表中。

### 3. 檢視日誌

**步驟**：
1. 點選選單欄 **幫助** → **日誌**
2. 或使用快捷鍵 `Ctrl+L`
3. 在日誌視窗中檢視程式執行日誌

### 4. 檢視已載入外掛

**步驟**：
1. 點選選單欄 **外掛** → **外掛管理**
2. 檢視已載入的外掛列表
3. 可以啟用/禁用外掛

## 常見首次執行問題

### 問題 1：程式啟動慢

**原因**：首次啟動需要初始化配置、掃描外掛等

**解決方法**：
- 正常情況下首次啟動需要 10-30 秒
- 如果超過 1 分鐘，檢查日誌檢視具體原因
- 可能是外掛載入問題，嘗試臨時移除 Plugins 目錄中的外掛

### 問題 2：介面顯示異常

**原因**：解析度或 DPI 設定不匹配

**解決方法**：
- 推薦使用 1920x1080 或更高解析度
- DPI 縮放設定為 100%
- 在 Windows 設定中調整顯示縮放

### 問題 3：找不到配置檔案

**原因**：權限問題或路徑問題

**解決方法**：
- 檢查使用者文件目錄是否可訪問
- 以管理員身份執行程式
- 檢查日誌中的錯誤資訊

### 問題 4：外掛載入失敗

**原因**：外掛依賴缺失或版本不匹配

**解決方法**：
- 檢視日誌中的外掛載入錯誤
- 檢查外掛的 manifest.json 檔案
- 確保外掛與 ColorVision 版本匹配

更多問題請參考：[故障排除](../01-user-guide/troubleshooting/common-issues.md)
