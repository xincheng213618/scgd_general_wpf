# 主視窗導覽

主視窗是 ColorVision 的工作臺。大多數日常操作都會從這裡進入，包括裝置、流程、影像檢視、日誌、搜尋和狀態監控。

## 進入主視窗後先看哪裡

### 頂部選單和搜尋區

- 選單是主要功能入口
- 搜尋框用於快速定位命令、頁面或物件
- 當你不知道功能藏在哪時，優先先用搜尋

### 左側導航區

- 常用於檢視工程、資源樹或相關檢視頁籤
- 如果當前頁面內容很多，先從左側結構切入通常更快

### 中間主工作區

- 顯示當前正在操作的核心內容
- 可能是影像檢視、流程視窗、模板視窗或其他業務介面
- 佈局可能根據當前工作模式切換為單檢視或多檢視

### 底部狀態列

- 用來觀察系統狀態、提示資訊和部分快捷入口
- 當裝置、服務或外掛狀態異常時，先注意這裡有沒有提示

## 你通常會在主視窗做什麼

- 開啟裝置相關視窗
- 進入流程設計和執行頁面
- 檢視影像或結果視窗
- 開啟日誌、終端和配置介面
- 搜尋功能入口或物件位置

## 建議的熟悉順序

1. 先認識主選單、搜尋區和狀態列。
2. 再理解左側導航和主工作區是如何配合的。
3. 接著進入 [屬性編輯器](./property-editor.md) 和 [影像編輯器概覽](../image-editor/overview.md)。
4. 涉及硬體時，繼續看 [裝置服務概覽](../devices/overview.md)。
5. 需要自動化時，進入 [工作流程](../workflow/README.md)。

## 常見問題

### 找不到某個功能入口

- 先用搜尋框搜關鍵詞
- 再回到對應章節首頁重新進入
- 如果是外掛功能，檢查外掛是否已經被載入

### 開啟的視窗太多，不知道當前在看什麼

- 回到左側導航重新確認當前物件
- 觀察主工作區標題和狀態列資訊
- 按業務重新進入一次對應頁面，而不是在多個視窗間盲切

### 狀態列沒有出現預期提示

- 先確認相關服務、裝置或外掛是否已經真正啟動
- 再檢視 [日誌檢視器](./log-viewer.md) 獲取更明確的資訊
