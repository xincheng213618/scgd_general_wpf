# 屬性編輯器

ColorVision 使用基於後設資料的屬性編輯器統一展示和編輯物件屬性。大多數模板、裝置和配置物件都會透過同一套 PropertyGrid 機制暴露參數。

## 你會看到什麼

- 左側或彈窗中的屬性列表
- 按分類分組的參數區域
- 針對不同型別自動切換的編輯控制元件，例如文字、列舉、布林值、顏色和檔案路徑

## 常見使用方式

1. 在物件樹、模板編輯器或裝置配置介面選中一個物件。
2. 在屬性面板中檢視分組後的參數。
3. 修改值後，觀察介面或物件狀態是否即時更新。
4. 如有重置或校驗按鈕，優先使用按鈕而不是直接手工改底層配置檔案。

## 這套編輯器依賴什麼

- 屬性上的 `Category`、`DisplayName`、`Description` 等後設資料
- 必要時使用自訂編輯器型別來決定控制元件樣式
- 部分複雜物件會在編輯器中展開為巢狀結構，而不是簡單字串
