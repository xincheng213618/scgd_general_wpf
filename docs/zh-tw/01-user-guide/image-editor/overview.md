# 影像編輯器概覽

影像編輯器是 ColorVision 裡最常用的結果檢視和互動區域之一。你會在這裡檢視影像、縮放平移、畫 ROI、做標註、播放影片，以及在特定場景下切到 3D 或偽彩色檢視。

## 你能在這裡做什麼

- 檢視常見影像結果
- 縮放、平移和適配畫面
- 畫矩形、圓、多邊形、曲線和文字標註
- 檢視屬性面板並調整所選物件
- 使用偽彩色、直方圖或 3D 檢視觀察資料
- 在影片檔案場景下直接播放、暫停、拖動進度和切換預覽縮放

## 常見使用順序

1. 先開啟一張影像或一個結果視窗。
2. 用縮放和平移確認關鍵區域。
3. 需要測量或標記時，再切到繪圖工具。
4. 需要看物件參數時，開啟屬性面板。
5. 如果輸入是影片，優先先確認播放是否流暢，再做進一步分析。

## 影片模式會發生什麼

當開啟的是支援的影片檔案時，影像編輯器會切到影片工作模式。此時通常會看到：

- 播放和暫停控制
- 進度條
- 倍速切換
- 預覽縮放切換
- 靜音或自動隱藏工具欄相關選項

如果是高解析度影片，先降低預覽縮放通常比一開始就追求原始解析度更實用。

## 哪些場景特別常見

### 看檢測結果

- 先確認原圖和結果覆蓋是否對應
- 再看 ROI、標註和屬性是否正確

### 調整影像顯示

- 優先使用縮放、偽彩色和對比觀察
- 不要一開始就修改底層配置，先確認是不是顯示問題

### 檢查影片或動態資料

- 先驗證播放是否正常
- 再決定是否降低預覽比例或關閉部分附加顯示

## 常見問題

### 影像顯示正常，但操作感覺卡頓

- 先縮小檢視或減少同時顯示的內容
- 影片場景下先降低預覽縮放
- 3D 檢視場景下先降低目標解析度或切回普通檢視確認問題範圍

### 標註或 ROI 不容易選中

- 先確認當前是否還停留在繪圖模式
- 嘗試回到選擇狀態後再點選物件
- 必要時結合 [屬性編輯器](../interface/property-editor.md) 檢視當前選中物件

### 影片畫面不同步或不流暢

- 先暫停後重新播放
- 再嘗試拖動進度條跳轉
- 對高解析度影片優先降低預覽縮放後複查

### 3D 檢視開啟後效果異常

- 先確認輸入資料本身是否適合做高度或強度視覺化
- 再檢查是否需要降低目標解析度或切換顏色對映
