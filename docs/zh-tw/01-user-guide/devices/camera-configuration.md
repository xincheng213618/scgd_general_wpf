# 相機參數配置

本頁講“相機服務配置頁”裡通常要改什麼。這裡調整的是邏輯相機服務參數，其中一部分預設值會來自繫結的物理相機。

## 常見配置內容

### 繫結資訊

- 當前服務例項名稱
- 繫結的物理相機或 CameraCode
- 當前專案裡真正被呼叫的是哪一臺相機服務

### 採集參數

- 通道和位深
- 拍攝模式
- 曝光、增益、飽和度
- 自動曝光開關和相關參數

### 擴充套件參數

- 影片相關配置
- 檔案快取和儲存格式
- 電機、CFW、自動對焦等從屬配置

## 建議調整順序

1. 先繫結正確的物理相機。
2. 讓服務同步一輪基礎參數。
3. 再調整曝光、增益、通道和模式。
4. 儲存後用實時影像或流程跑一次驗證。
5. 需要高階調優時，再進入模板或校準頁。

## 哪些參數最容易互相影響

- 相機模式和通道
- 位深與採集模式
- 曝光時間和增益
- 自動曝光與手動曝光參數
- 物理相機預設值與當前服務顯示配置

如果某次修改之後畫面明顯異常，優先檢查這幾組是否一起變了，而不是隻盯著單個參數。

## 常見問題

### 參數改了沒有效果

- 確認改的是當前正在使用的那臺相機服務
- 儲存後重新開啟檢視或重啟服務
- 檢查是否被物理相機同步配置覆蓋

### 曝光或通道設定不合理

- 先確認當前相機模式支援哪些通道
- 三通道和單通道參數不要混用
- 出現過曝或噪點時，優先回撥曝光和增益

### 儲存後流程結果仍異常

- 用實時畫面先驗證採圖是否正常
- 再檢查流程裡的模板參數是否仍是舊值
- 必要時聯動檢查校準和自動曝光模板
