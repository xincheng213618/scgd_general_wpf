## 外掛開發指南

### IPlugin 介面
```csharp
public interface IPlugin
{
    string Name { get; }
    string Version { get; }
    void Initialize();
    void Shutdown();
}
```

### 外掛生命週期
1. 程式啟動時自動發現外掛
2. 呼叫 Initialize() 方法初始化
3. 程式關閉時呼叫 Shutdown() 方法清理資源
