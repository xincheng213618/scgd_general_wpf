# 图像编辑器概览

图像编辑器是 ColorVision 里最常用的结果查看和交互区域之一。你会在这里查看图像、缩放平移、画 ROI、做标注、播放视频，以及在特定场景下切到 3D 或伪彩色视图。

## 你能在这里做什么

- 查看常见图像结果
- 缩放、平移和适配画面
- 画矩形、圆、多边形、曲线和文本标注
- 查看属性面板并调整所选对象
- 使用伪彩色、直方图或 3D 视图观察数据
- 在视频文件场景下直接播放、暂停、拖动进度和切换预览缩放

## 常见使用顺序

1. 先打开一张图像或一个结果窗口。
2. 用缩放和平移确认关键区域。
3. 需要测量或标记时，再切到绘图工具。
4. 需要看对象参数时，打开属性面板。
5. 如果输入是视频，优先先确认播放是否流畅，再做进一步分析。

## 视频模式会发生什么

当打开的是支持的视频文件时，图像编辑器会切到视频工作模式。此时通常会看到：

- 播放和暂停控制
- 进度条
- 倍速切换
- 预览缩放切换
- 静音或自动隐藏工具栏相关选项

如果是高分辨率视频，先降低预览缩放通常比一开始就追求原始分辨率更实用。

## 哪些场景特别常见

### 看检测结果

- 先确认原图和结果覆盖是否对应
- 再看 ROI、标注和属性是否正确

### 调整图像显示

- 优先使用缩放、伪彩色和对比观察
- 不要一开始就修改底层配置，先确认是不是显示问题

### 检查视频或动态数据

- 先验证播放是否正常
- 再决定是否降低预览比例或关闭部分附加显示

## 常见问题

### 图像显示正常，但操作感觉卡顿

- 先缩小视图或减少同时显示的内容
- 视频场景下先降低预览缩放
- 3D 视图场景下先降低目标分辨率或切回普通视图确认问题范围

### 标注或 ROI 不容易选中

- 先确认当前是否还停留在绘图模式
- 尝试回到选择状态后再点击对象
- 必要时结合 [属性编辑器](../interface/property-editor.md) 查看当前选中对象

### 视频画面不同步或不流畅

- 先暂停后重新播放
- 再尝试拖动进度条跳转
- 对高分辨率视频优先降低预览缩放后复查

### 3D 视图打开后效果异常

- 先确认输入数据本身是否适合做高度或强度可视化
- 再检查是否需要降低目标分辨率或切换颜色映射
