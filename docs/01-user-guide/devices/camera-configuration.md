# 相机参数配置

本页讲“相机服务配置页”里通常要改什么。这里调整的是逻辑相机服务参数，其中一部分默认值会来自绑定的物理相机。

## 常见配置内容

### 绑定信息

- 当前服务实例名称
- 绑定的物理相机或 CameraCode
- 当前项目里真正被调用的是哪一台相机服务

### 采集参数

- 通道和位深
- 拍摄模式
- 曝光、增益、饱和度
- 自动曝光开关和相关参数

### 扩展参数

- 视频相关配置
- 文件缓存和保存格式
- 电机、CFW、自动对焦等从属配置

## 建议调整顺序

1. 先绑定正确的物理相机。
2. 让服务同步一轮基础参数。
3. 再调整曝光、增益、通道和模式。
4. 保存后用实时图像或流程跑一次验证。
5. 需要高级调优时，再进入模板或校准页。

## 哪些参数最容易互相影响

- 相机模式和通道
- 位深与采集模式
- 曝光时间和增益
- 自动曝光与手动曝光参数
- 物理相机默认值与当前服务显示配置

如果某次修改之后画面明显异常，优先检查这几组是否一起变了，而不是只盯着单个参数。

## 常见问题

### 参数改了没有效果

- 确认改的是当前正在使用的那台相机服务
- 保存后重新打开视图或重启服务
- 检查是否被物理相机同步配置覆盖

### 曝光或通道设置不合理

- 先确认当前相机模式支持哪些通道
- 三通道和单通道参数不要混用
- 出现过曝或噪点时，优先回调曝光和增益

### 保存后流程结果仍异常

- 用实时画面先验证采图是否正常
- 再检查流程里的模板参数是否仍是旧值
- 必要时联动检查校准和自动曝光模板
