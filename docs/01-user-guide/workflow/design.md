# 流程设计

介绍如何使用可视化编辑器设计工作流程。

## 打开流程编辑器

1. 在主菜单选择"流程" → "流程编辑器"
2. 或使用快捷键 `Ctrl+F`

## 添加节点

### 从工具箱拖拽

1. 在左侧工具箱选择节点类型
2. 拖拽到画布上
3. 配置节点属性

### 常用节点类型

**设备控制节点**:
- 相机采集
- 电机移动
- 光谱测量

**算法处理节点**:
- 图像处理
- 缺陷检测
- 颜色分析

**逻辑控制节点**:
- 条件判断
- 循环
- 延时

## 连接节点

1. 点击源节点的输出端口
2. 拖拽到目标节点的输入端口
3. 释放鼠标完成连接

## 配置节点

1. 双击节点或在属性面板查看
2. 修改参数配置
3. 保存更改

## 保存流程

- **保存**: `Ctrl+S`
- **另存为**: `Ctrl+Shift+S`
- **导出模板**: 右键菜单 → "导出为模板"

## 最佳实践

1. 使用有意义的节点名称
2. 添加注释说明复杂逻辑
3. 模块化设计，创建可重用的子流程
4. 测试每个节点后再连接
