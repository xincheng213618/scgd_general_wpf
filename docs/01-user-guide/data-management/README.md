# 数据管理

ColorVision 提供完整的数据管理功能，支持数据库存储和文件管理。

## 概述

数据管理系统负责：

- 📊 测试数据存储
- 🗄️ 数据库操作
- 📁 文件管理
- 📤 数据导入导出

## 数据库支持

ColorVision 支持多种数据库：

- **MySQL**: 生产环境推荐
- **SQLite**: 开发和小规模使用

详见：[数据库操作](./database.md)

## 文件管理

- 解决方案管理
- 图像文件预览
- 批量操作
- 文件搜索和过滤

## 数据导入导出

支持多种格式：

- Excel (xlsx)
- CSV
- JSON
- XML

详见：[数据导出与导入](./export-import.md)

## 数据备份

- 自动备份
- 手动备份
- 备份恢复
