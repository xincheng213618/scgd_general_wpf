# 数据导出与导入

介绍如何导出和导入数据。

## 数据导出

### 导出为 Excel

1. 选择要导出的数据
2. 右键菜单 → "导出" → "Excel"
3. 选择保存位置
4. 配置导出选项
5. 点击"导出"

### 导出为 CSV

1. 选择数据
2. 导出 → CSV
3. 选择分隔符（逗号、制表符等）
4. 保存文件

### 导出为 JSON

适合程序处理的格式：

```json
{
  "test_results": [
    {
      "id": 1,
      "name": "Test 1",
      "result": "Pass",
      "timestamp": "2024-01-01T10:00:00"
    }
  ]
}
```

## 数据导入

### 从 Excel 导入

1. 打开"数据" → "导入"
2. 选择 Excel 文件
3. 映射列到数据字段
4. 预览数据
5. 点击"导入"

### 从 CSV 导入

1. 选择 CSV 文件
2. 指定分隔符
3. 配置列映射
4. 导入数据

### 批量导入

- 支持导入整个文件夹
- 自动识别文件格式
- 进度显示

## 导入导出设置

### 字段映射

配置源字段到目标字段的映射关系。

### 数据验证

- 类型检查
- 必填字段验证
- 唯一性检查

### 错误处理

- 跳过错误行
- 记录错误日志
- 部分导入

## 模板管理

### 保存导出模板

保存常用的导出配置为模板，方便重复使用。

### 使用模板

1. 选择已保存的模板
2. 直接导出，无需重新配置
