# 数据导出与导入

当前仓库里并没有一个统一的“所有数据都从这里导入导出”的总控中心。更接近真实情况的是：不同对象走不同入口，设置、流程模板和具体结果数据各自有自己的导入导出方式。

## 先怎么理解这一页

如果你要做导入导出，先回答三个问题：

1. 你要处理的是设置、流程模板，还是某个结果模块的数据？
2. 你需要的是全局配置迁移，还是单个业务对象导出？
3. 这个功能是不是本来就属于某个具体窗口，而不是数据管理中心？

## 当前能明确证实的入口

### 设置导入导出

当前有明确菜单入口：

- 工具 → 导入导出设置

这里至少包含两类动作：

- 导出设置到 `.cvsettings`
- 从 `.cvsettings` 导入设置

如果你的目标是迁移软件配置，而不是迁移结果数据，优先从这里开始。

### 流程模板导入导出

流程模板的导入导出不在数据管理页统一处理，而是在流程设计器中单独提供：

- 导出当前流程
- 导入流程
- 导入模块

如果你要迁移流程内容，优先去 [流程设计](../workflow/design.md)。

### 模块内结果导出

一些业务窗口会提供自己的导出功能，例如：

- 流程节点分析窗口可导出 CSV
- 部分插件或图像/测量窗口有自己的 CSV 或图像导出入口

这类导出通常跟具体业务对象强绑定，不应该再被描述成统一的全局“数据导出中心”。

## 对象和入口对照

| 要交付的对象 | 优先入口 | 交付前确认 |
| --- | --- | --- |
| 软件设置 | 工具 → 导入导出设置 | `.cvsettings` 能导入，重启后关键设置仍在 |
| 流程模板 | 流程设计器导入/导出 | 导入后起始节点、设备绑定、模板参数正确 |
| 数据库记录 | 数据库浏览器或业务结果页 | 能按 SN、时间、批次查到同一轮记录 |
| CSV/Excel | 对应业务窗口或插件导出 | 字段顺序、单位、PASS/FAIL、编码符合客户要求 |
| PDF/报告 | 项目窗口或插件自身报告入口 | 页眉、客户标识、结果截图、判定项正确 |
| 图片/overlay | 图像编辑器或结果窗口 | 原图、ROI/POI、标注坐标和文件名能对应 |
| Socket/MES 响应 | 项目窗口、SocketProtocol 或外部联调工具 | 请求和响应样例已保存，状态码和 Data 字段正确 |

## 导出交付前验收

导出功能最容易出现“按钮能点，但交出去的内容不对”。交付前至少做一次端到端核对。

| 步骤 | 操作 | 通过标准 |
| --- | --- | --- |
| 1 | 用一组明确的 SN 或测试批次跑一次最小流程 | 后续查询、导出和外部响应都使用同一组标识 |
| 2 | 在数据库或结果窗口确认源数据存在 | 源数据不是空表，也不是旧批次 |
| 3 | 从目标窗口执行导出 | 文件成功生成，路径和文件名可解释 |
| 4 | 打开导出文件核对字段 | 字段顺序、单位、判定、时间、SN 与客户要求一致 |
| 5 | 保存一组样例文件和截图 | 后续版本升级后可按样例复测 |

## 导出失败分层定位

| 现象 | 先查 | 再查 |
| --- | --- | --- |
| 导出按钮找不到 | 对象是否属于设置、流程或某个业务窗口 | 插件/项目文档是否声明支持导出 |
| 文件生成但为空 | 源数据是否已落库，批次/SN 是否选对 | 导出筛选条件和字段映射 |
| 字段缺失或顺序不对 | 当前窗口导出的对象是否正确 | 项目 exporter、客户格式版本 |
| 图片或 overlay 不对齐 | 结果图和原图是否同一轮 | ROI/POI 坐标、图像缩放或模板版本 |
| 外部系统收不到 | ColorVision 是否已完成并生成结果 | 协议、端口、项目 handler、响应字段 |
| 迁移后行为变化 | 导出的是否只是设置而不是结果数据 | 旧配置、流程模板、数据库备份是否同步 |

## 常见使用顺序

1. 先确认你要导出的到底是什么对象。
2. 如果是全局设置，走“导入导出设置”。
3. 如果是流程内容，走 [流程设计](../workflow/design.md) 里的导入导出。
4. 如果是结果数据或业务数据，先回到对应模块窗口找它自己的导出入口。
5. 真正涉及库内数据时，再配合 [数据库操作](./database.md) 确认源数据范围。

## 当前这页不再承诺什么

以下这些能力，当前使用手册页不再默认宣称为统一可用能力，除非你已经在具体模块窗口里看到了它：

- 统一 Excel 导出中心
- 统一 JSON 导出中心
- 统一 XML 导出中心
- 统一 PDF 报告导出中心
- 通用列映射导入向导
- 通用批量文件夹导入向导

如果某个插件或窗口确实支持这些格式，应在那个模块自己的页面说明，而不是放在这里泛讲。

## 常见问题

### 我想导出数据，但找不到统一入口

- 先不要继续找总菜单
- 先确认对象属于设置、流程还是某个业务结果窗口
- 再回到对应模块页查找导出入口

### 导出了设置，但业务结果没有跟着迁移

- `.cvsettings` 主要用于配置迁移，不等于数据库结果迁移
- 真正的结果数据要结合 [数据库操作](./database.md) 或对应业务模块单独处理

### 流程导入成功了，但执行结果不对

- 先确认导入的是不是正确流程版本
- 再回到 [流程执行与调试](../workflow/execution.md) 验证依赖设备和模板
- 必要时重新检查模块导入和流程参数
