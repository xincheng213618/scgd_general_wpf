项目概览

ColorVision 是由视彩（上海）光电技术有限公司开发的一款基于 Windows WPF 技术的专业应用程序，专注于提供先进的色彩管理和光电技术解决方案。

核心功能

  • 设备集成：支持多种物理设备的配置与控制，包括相机、校准设备、算法模块、光谱仪等
  • 流程自动化：基于流程引擎的任务调度系统，支持复杂流程的自动化执行
  • 模板管理：完整的模板生命周期管理，支持创建、编辑、导入导出和快速检测
  • 用户体验：多语言、多主题支持，热键管理，符合国际化需求
  • 扩展性：插件架构设计，支持第三方功能扩展
  • 项目管理：工程项目管理，支持数据归档和导出
  • 自动更新：内置自动和手动软件更新机制

技术栈

  • 框架：.NET 8.0, Windows WPF
  • UI 库：HandyControl, Extended.Wpf.Toolkit, WPF-UI
  • 图像处理：OpenCV
  • 通信协议：MQTT
  • 数据存储：数据库持久化
  • 日志系统：log4net, spdlog

支持环境

  • 操作系统：Windows 10 1903+ / Windows 11
  • 分辨率：1920x1080 及以上
  • 运行时：.NET 8.0 Runtime
