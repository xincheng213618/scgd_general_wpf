## 🚀 快速开始

如果您是新的开发者，建议按以下顺序学习：

1. **了解架构** - 查看[架构设计](../03-architecture/README.md)
2. **配置开发环境** - 安装必要的工具和依赖
3. **阅读核心概念** - 理解扩展性和插件机制
4. **开发第一个插件** - 跟随[插件开发入门](./plugin-development/getting-started.md)
