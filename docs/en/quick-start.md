# Quick Start

This page is a compact English entry. For complete and current procedures, use the Simplified Chinese documentation.

## Install And Start

1. Prepare a Windows environment with the required .NET SDK/runtime and device dependencies.
2. Build or install ColorVision from the release package used by your delivery team.
3. Start the main WPF application.
4. Open the main window, confirm the device list, logs, and project/plugin entries.
5. For first-run details, see the Chinese [quick start](/00-getting-started/quick-start), [installation guide](/00-getting-started/installation), and [first run](/00-getting-started/first-steps).
