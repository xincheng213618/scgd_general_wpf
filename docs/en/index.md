## Documentation Policy

ColorVision now uses Simplified Chinese as the complete source of truth for product, project, plugin, and source-reference documentation.

English pages are intentionally compact. They help non-Chinese readers find the installation path, development entry, GitHub repository, changelog, and license without duplicating every internal handoff page.

Traditional Chinese, Japanese, and Korean translation copies were removed from the current working tree. They can be recovered from Git history if a concrete delivery need appears.
