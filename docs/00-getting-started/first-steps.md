# 首次运行指南

本指南帮助您在首次安装 ColorVision 后，了解如何启动程序并进行基本配置。

## 启动程序

### 方式一：从开始菜单启动

1. 打开 Windows 开始菜单
2. 找到 "ColorVision" 或 "视彩光电"
3. 点击图标启动程序

### 方式二：从桌面快捷方式启动

如果安装时选择了创建桌面快捷方式，双击桌面上的 ColorVision 图标即可启动。

### 方式三：从命令行启动（开发者）

```bash
cd "安装目录"
.\ColorVision.exe
```

## 首次启动时的自动配置

程序首次启动时会自动执行以下操作：

1. **创建配置文件**
   - 位置：`%USERPROFILE%\Documents\ColorVision\`
   - 包含：用户配置、设备配置、流程配置等

2. **初始化日志系统**
   - 日志目录：`%USERPROFILE%\Documents\ColorVision\logs\`
   - 日志级别：默认为 Info

3. **扫描插件**
   - 自动扫描 `Plugins` 目录
   - 加载所有可用插件

4. **显示主窗口**
   - 展示主界面
   - 显示欢迎提示（可选）

## 基本操作演示

### 1. 打开图像

**步骤**：
1. 点击菜单栏 **文件** → **打开图像**
2. 或使用快捷键 `Ctrl+O`
3. 选择图像文件（支持 BMP, JPG, PNG, TIFF 等格式）
4. 点击"打开"

图像将在中央显示区域显示。

### 2. 添加模拟设备

**步骤**：
1. 点击菜单栏 **设备** → **添加设备**
2. 选择 **模拟相机** 或其他模拟设备
3. 配置设备参数
4. 点击"确定"

设备将出现在左侧设备列表中。

### 3. 查看日志

**步骤**：
1. 点击菜单栏 **帮助** → **日志**
2. 或使用快捷键 `Ctrl+L`
3. 在日志窗口中查看程序运行日志

### 4. 查看已加载插件

**步骤**：
1. 点击菜单栏 **插件** → **插件管理**
2. 查看已加载的插件列表
3. 可以启用/禁用插件

## 常见首次运行问题

### 问题 1：程序启动慢

**原因**：首次启动需要初始化配置、扫描插件等

**解决方法**：
- 正常情况下首次启动需要 10-30 秒
- 如果超过 1 分钟，检查日志查看具体原因
- 可能是插件加载问题，尝试临时移除 Plugins 目录中的插件

### 问题 2：界面显示异常

**原因**：分辨率或 DPI 设置不匹配

**解决方法**：
- 推荐使用 1920x1080 或更高分辨率
- DPI 缩放设置为 100%
- 在 Windows 设置中调整显示缩放

### 问题 3：找不到配置文件

**原因**：权限问题或路径问题

**解决方法**：
- 检查用户文档目录是否可访问
- 以管理员身份运行程序
- 检查日志中的错误信息

### 问题 4：插件加载失败

**原因**：插件依赖缺失或版本不匹配

**解决方法**：
- 查看日志中的插件加载错误
- 检查插件的 manifest.json 文件
- 确保插件与 ColorVision 版本匹配

更多问题请参考：[故障排除](../01-user-guide/troubleshooting/common-issues.md)
