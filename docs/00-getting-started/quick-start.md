# 快速上手

本页面向第一次接触 ColorVision 的读者，目标是在最短路径内完成一次“启动程序 -> 识别界面 -> 做一个基本操作 -> 知道后面该看哪里”的闭环。

## 开始之前

建议先完成以下准备：

1. 已确认 [系统要求](./prerequisites.md)
2. 已完成 [安装指南](./installation.md)
3. 可以正常启动主程序

## 10 分钟体验路线

### 第 1 步：启动并确认主界面正常

- 启动 ColorVision
- 确认主窗口可以正常显示
- 如果首次启动较慢，先等待初始化和插件扫描完成

想先熟悉界面结构，可以直接阅读 [主窗口导览](../01-user-guide/interface/main-window.md)。

### 第 2 步：认识几个核心区域

建议先只关注这几个区域：

- 菜单栏：进入文件、设备、流程、帮助等主要功能
- 中央工作区：查看图像、流程或结果
- 属性面板：查看和修改当前对象参数
- 状态栏与日志：确认程序是否报错、当前状态是否正常

### 第 3 步：做一次最小操作

如果你只是想确认程序可用，优先选下面两种最简单的路径之一：

#### 路径 A：打开一张图片

1. 从文件菜单打开本地图片
2. 确认图像能在工作区显示
3. 观察属性面板或相关工具是否正常响应

#### 路径 B：添加一个模拟或现有设备

1. 进入设备相关菜单
2. 添加一个模拟设备，或连接当前已有设备
3. 确认设备能出现在设备区域并正常响应

设备相关内容可以继续看 [设备服务概览](../01-user-guide/devices/overview.md)。

### 第 4 步：尝试一个更接近真实业务的入口

根据你的目标选择后续阅读：

- 想做图像查看和标注：前往 [图像编辑器概览](../01-user-guide/image-editor/overview.md)
- 想配置设备和采集链路：前往 [设备服务概览](../01-user-guide/devices/overview.md)
- 想开始理解流程如何设计和执行：前往 [工作流程概览](../01-user-guide/workflow/README.md)
