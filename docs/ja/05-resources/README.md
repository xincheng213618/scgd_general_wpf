#付録とリソース

この部分には、索引または付録として適した長期安定したコンテンツのみが保持され、散在したトピックの説明は含まれなくなりました。

## 現在予約されているコンテンツ

- [モジュールとドキュメントの比較表](./project-struction/module-documentation-map.md)
- [ソフトウェア使用許諾契約](./legal/software-agreement.md)
- [CHANGELOG.md](https://github.com/xincheng213618/scgd_general_wpf/blob/master/CHANGELOG.md)

## 使用法

- ウェアハウスの構造を理解したい場合：まず [プロジェクト構造の概要](./project-structor/README.md) を読んでください。
- モジュールに対応するドキュメント エントリをすばやく見つけたい場合: [モジュールとドキュメントの比較表](./project- Structure/module-documentation-map.md) を参照してください。
- バージョン変更を確認したい場合: ウェアハウスのルートディレクトリにある`CHANGELOG.md`を直接確認してください。
- 法的テキストを表示するには: [ソフトウェア使用許諾契約書](./legal/software-agreement.md) と入力します。

## この配置の境界

- 古い API v1.1、テンプレート、データ ストレージ、その他のエントリは、リソース領域のナビゲーションに配置されなくなります。
- テーマの内容は、展開については開発ガイドに、インターフェイスについては API リファレンスに、システム設計についてはアーキテクチャ設計に戻すなど、対応する章に戻す必要があります。
