## ソースコードから始める

現在のウェアハウスは主に Windows WPF と x64 用です。まず依存関係の回復を完了してから、メイン プログラムをビルドすることをお勧めします。


```powershell
dotnet restore
dotnet build -p:Platform=x64
dotnet run --project ColorVision/ColorVision.csproj
```
