# 初回実行ガイド

このガイドは、ColorVision を初めてインストールした後にプログラムを起動し、基本的な構成を実行する方法を学習するのに役立ちます。

## プログラムを開始します

### 方法 1: スタート メニューから起動する

1. Windows の [スタート] メニューを開きます
2.「ColorVision」または「ColorVision」を検索します。
3. アイコンをクリックしてプログラムを起動します

### 方法 2: デスクトップのショートカットから開始する

インストール中にデスクトップ ショートカットを作成することを選択した場合は、デスクトップ上の ColorVision アイコンをダブルクリックして起動します。

### 方法 3: コマンド ラインから開始する (開発者)


```powershell
cd "安装目录"
.\ColorVision.exe
```


## 初回起動時の自動構成

プログラムは、初回起動時に次の操作を自動的に実行します。

1. **設定ファイルの作成**
   - 場所: `%USERPROFILE%\Documents\ColorVision\`
   - ユーザー構成、デバイス構成、プロセス構成などが含まれます。

2. **ログ システムを初期化します**
   - ログディレクトリ: `%USERPROFILE%\Documents\ColorVision\logs\`
   - ログレベル: デフォルトは情報です

3. **スキャン プラグイン**
   - `Plugins` ディレクトリを自動的にスキャンします
   - 利用可能なすべてのプラグインをロードします

4. **メインウィンドウを表示**
   - メインインターフェイスを表示します
   - ようこそプロンプトを表示する (オプション)

### メインエリアの説明

- **メニュー バー**: ファイル、編集、表示、デバイス、プロセス、プラグイン、ヘルプ、その他のメニュー
- **ツールバー**: よく使用される機能のショートカット ボタン
- **デバイスリスト**: 接続されているデバイスと利用可能なデバイスを表示します
- **画像表示領域**: 画像、プロセスエディターなどを表示します。
- **プロパティ パネル**: 選択したオブジェクトのプロパティを表示します。
- **ステータス バー**: プログラムのステータスとプロンプト情報を表示します。

インターフェイスの詳細な説明については、[メイン ウィンドウ ガイド](../01-user-guide/interface/main-window.md) を参照してください。

## 基本操作のデモンストレーション

### 1. 画像を開きます

**手順**:
1. メニューバーの [ファイル] → [画像を開く] をクリックします。
2. または、ショートカット キー `Ctrl+O` を使用します。
3. 画像ファイルを選択します (BMP、JPG、PNG、TIFF などの形式をサポート)
4.「開く」をクリックします

画像は中央の表示領域に表示されます。

### 2. シミュレートされたデバイスを追加します

**手順**:
1. メニュー バーの [**デバイス**] → [**デバイスの追加**] をクリックします。
2. **アナログ カメラ**またはその他のアナログ デバイスを選択します
3. デバイスパラメータを設定する
4.「OK」をクリックします。

デバイスが左側のデバイスリストに表示されます。

### 3. ログを表示する

**手順**:
1. メニュー バーの [**ヘルプ**] → [**ログ**] をクリックします。
2. または、ショートカット キー `Ctrl+L` を使用します。
3. ログウィンドウでプログラム実行ログを表示します。

### 4. ロードされたプラグインを表示する

**手順**:
1. メニュー バーの [**プラグイン**] → [**プラグイン管理**] をクリックします。
2. ロードされているプラグインのリストを表示する
3. プラグインを有効/無効にできます

## 初回実行時の一般的な問題

### 問題 1: プログラムの起動が遅い

**理由**: 最初の起動には初期化設定、スキャン プラグインなどが必要です。

**解決策**:
- 通常、最初の起動には 10 ～ 30 秒かかります
- 1 分以上かかる場合は、ログを確認して具体的な理由を確認してください。
- プラグインの読み込みに問題がある可能性があります。Plugins ディレクトリ内のプラグインを一時的に削除してみてください。

### 問題 2: インターフェイスの表示が異常である

**原因**: 解像度または DPI 設定が一致しません

**解決策**:
- 1920x1080 以上の解像度を推奨
- DPI スケーリングを 100% に設定
- Windows 設定で表示スケーリングを調整します

### 問題 3: 構成ファイルが見つかりません

**原因**: 権限の問題またはパスの問題

**解決策**:
- ユーザードキュメントディレクトリにアクセスできるかどうかを確認してください
- 管理者としてプログラムを実行します
- ログでエラー メッセージを確認します。

### 問題 4: プラグインのロードに失敗しました

**原因**: プラグインの依存関係が欠落しているか、バージョンが一致していません

**解決策**:
- プラグイン読み込みエラーのログを確認する
- プラグインのmanifest.jsonファイルを確認してください
- プラグインが ColorVision のバージョンと一致していることを確認してください

その他の質問については、[トラブルシューティング](../01-user-guide/troubleshooting/common-issues.md) を参照してください。
