#プロパティエディター

ColorVision は、メタデータベースのプロパティ エディタを使用して、オブジェクト プロパティを均一に表示および編集します。ほとんどのテンプレート、デバイス、構成オブジェクトは、同じ PropertyGrid メカニズムを通じてパラメーターを公開します。

## 何が見えるでしょうか

- 左側またはポップアップウィンドウのプロパティリスト
- カテゴリごとにグループ化されたパラメーター領域
- テキスト、列挙型、ブール値、色、ファイル パスなど、さまざまなタイプに応じて自動的に切り替わる編集コントロール

## 一般的な使用法

1. オブジェクト ツリー、テンプレート エディター、またはデバイス設定インターフェイスでオブジェクトを選択します。
2. プロパティ パネルでグループ化されたパラメータを表示します。
3. 値を変更した後、インターフェイスまたはオブジェクトのステータスがすぐに更新されるかどうかを確認します。
4. リセットまたは検証ボタンがある場合は、基礎となる構成ファイルを手動で直接変更するのではなく、ボタンの使用を優先します。

## このエディタは何に依存していますか?

- `Category`、`DisplayName`、`Description` などの属性のメタデータ
- 必要に応じてカスタム エディター タイプを使用してコントロール スタイルを決定します
- 一部の複雑なオブジェクトは、単純な文字列ではなく、エディターでネストされた構造に展開されます。
