# 画像エディタの概要

画像エディタは、ColorVision で結果を表示および操作するために最も一般的に使用される領域の 1 つです。ここでは、画像の表示、ズームとパン、ROI の描画、注釈の作成、ビデオの再生、および特定のシナリオでの 3D または擬似カラー ビューへの切り替えを行うことができます。

## ここで何ができるでしょうか?

- 一般的な画像結果を表示
- 画面をズーム、パン、フィットさせます
- 長方形、円、多角形、曲線、テキスト注釈を描画します
- プロパティ パネルを表示し、選択したオブジェクトを調整します
- 偽色、ヒストグラム、または 3D ビューを使用してデータを観察
- ビデオ ファイル シーンで直接、再生、一時停止、進行状況のドラッグ、プレビュー ズームの切り替えが可能

## 一般的な使用シーケンス

1. まず、画像または結果ウィンドウを開きます。
2. ズームとパンを使用して重要な領域を特定します。
3. 測定またはマークを付ける必要がある場合は、描画ツールに切り替えます。
4. オブジェクトのパラメータを確認する必要がある場合は、プロパティ パネルを開きます。
5. 入力がビデオの場合は、さらに分析する前に、まず再生がスムーズかどうかを確認します。

## ビデオモードで何が起こるか

サポートされているビデオ ファイルを開くと、画像エディタはビデオ作業モードに切り替わります。この時点で、通常は次のように表示されます。

- 再生と一時停止のコントロール
- プログレスバー
- 倍速切り替え
- プレビューズーム切り替え
- ツールバー関連オプションをミュートまたは自動的に非表示にする

高解像度ビデオの場合、多くの場合、最初からネイティブ解像度を使用するよりも、最初にプレビュー ズームを下げる方が現実的です。

## 特に一般的なシナリオはどれですか?

### テスト結果を参照

- まず、元の画像と結果のカバレッジが一致しているかどうかを確認します。
- ROI、注釈、属性が正しいかどうかを再確認します

### 画像表示を調整する

- ズーム、偽色、コントラスト観察の使用を優先します。
- 最初から基礎となる設定を変更せず、最初に表示に問題があるかどうかを確認します。

### ビデオまたは動的データを確認する

- まずは正常に再生できるか確認してください
- プレビュー率を下げるか、追加の表示をオフにするかを決定します

## よくある質問

### 画像は正常に表示されるが、動作がもたつく感じがする

- ビューを縮小するか、同時に表示される内容を減らすことから始めます。
- ビデオシーンでは、最初にプレビューのズームを下げます
- 3D ビューのシナリオでは、まずターゲットの解像度を下げるか、通常のビューに戻して、問題の範囲を確認します。

### ラベルまたは ROI を選択するのは簡単ではありません

- 最初にまだ描画モードであるかどうかを確認してください
- オブジェクトをクリックする前に、選択状態に戻ってみてください。
- 必要に応じて、[プロパティ エディタ](../interface/property-editor.md) を使用して、現在選択されているオブジェクトを表示します。

### ビデオ画像が同期していないか、滑らかではありません。

- 一時停止してから再開します
- プログレスバーをドラッグして再度ジャンプしてみてください
- ズーム後のレビュー用に低解像度のプレビュービデオを優先します

### 3D ビューを開いた後の効果が異常です

- まず入力データ自体が高さまたは強度の可視化に適しているかどうかを確認します
- ターゲットの解像度を下げる必要があるか、カラー マッピングを切り替える必要があるか再確認してください。
