## プラグイン開発ガイド

### IPlugin インターフェース

```csharp
public interface IPlugin
{
    string Name { get; }
    string Version { get; }
    void Initialize();
    void Shutdown();
}
```


### プラグインのライフサイクル
1. プログラムの起動時にプラグインを自動的に検出します
2. Initialize() メソッドを呼び出して初期化します。
3. プログラムを閉じるときに Shutdown() メソッドを呼び出してリソースをクリーンアップします。
