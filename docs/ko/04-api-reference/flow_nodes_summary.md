# Flow Engine 노드 문서 요약

## 문서 설명

이 디렉터리에는 다음과 같이 자동으로 생성된 노드 문서가 포함되어 있습니다.

1. **`flow_nodes_reference.md`**(노드 참조 구성됨)
   - `NodeConfigurator` 디렉터리에 있는 구성 파일을 기반으로 합니다.
   - **42개의 구성된 노드**에 대한 자세한 속성이 포함되어 있습니다.
   - 각 노드에는 다음이 포함됩니다.
     - 구성 패널 속성(NodeConfigurator에서)
     - 클래스 수준 속성(FlowEngineLib 구현에서)
     - 기본 클래스, 구현 파일 및 기타 정보

2. **`flow_nodes_complete.md`** (전체 노드 목록)
   - `FlowEngineLib` 디렉토리의 모든 노드 클래스 정의를 기반으로 합니다.
   - **90개 노드 클래스**의 전체 목록
   - 유형별 통계
   - 각 노드에는 기본 클래스, 구현 파일 및 속성 수가 포함됩니다.

## 사용 제안

- **개발자**: 사용 가능한 모든 노드 클래스와 해당 속성은 `flow_nodes_complete.md`를 참조하세요.
- **구성자**: 구성된 노드의 패널 속성 및 구성 방법에 대해 알아보려면 `flow_nodes_reference.md`를 참조하세요.
- **관리자**: 두 문서를 함께 사용하여 노드 구성과 구현 간의 해당 관계를 이해합니다.
