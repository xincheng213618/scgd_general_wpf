## 플러그인 개발 가이드

### I플러그인 인터페이스
```csharp
공개 인터페이스 IPlugin
{
    문자열 이름 { get; }
    문자열 버전 { get; }
    무효 초기화();
    무효 종료();
}
```
### 플러그인 수명 주기
1. 프로그램 시작 시 자동으로 플러그인 검색
2. 초기화() 메서드를 호출하여 초기화합니다.
3. 프로그램이 닫힐 때 Shutdown() 메서드를 호출하여 리소스를 정리합니다.
