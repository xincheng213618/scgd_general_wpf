# 빨리 시작하세요

이 페이지는 ColorVision을 처음 접하는 독자를 위한 페이지입니다. 목표는 최단 경로에서 "프로그램 시작 -> 인터페이스 식별 -> 기본 작업 수행 -> 다음 위치 파악"의 폐쇄 루프를 완료하는 것입니다.

## 시작하기 전에

먼저 다음 준비를 완료하는 것이 좋습니다.

1. [시스템 요구사항](./prerequisites.md) 확인
2. [설치 가이드](./installation.md) 완료
3. 메인 프로그램을 정상적으로 시작할 수 있습니다

##10분 체험코스

### 1단계: 시작하고 기본 인터페이스가 정상인지 확인합니다.

- ColorVision 시작
- 메인창이 정상적으로 표시되는지 확인
- 처음 시작이 느린 경우 먼저 초기화 및 플러그인 검색이 완료될 때까지 기다리세요.

먼저 인터페이스 구조를 익히고 싶다면 [메인 윈도우 가이드](../01-user-guide/interface/main-window.md)를 직접 읽어보시면 됩니다.

### 2단계: 몇 가지 핵심 영역 알아보기

먼저 다음 영역에 집중하는 것이 좋습니다.

- 메뉴바 : 파일, 장치, 프로세스, 도움말 등 주요 기능을 입력합니다.
- 중앙 작업 공간: 이미지, 프로세스 또는 결과 보기
- 속성 패널: 현재 개체 매개변수 보기 및 수정
- 상태 표시줄 및 로그: 프로그램 오류 보고 여부, 현재 상태가 정상인지 확인

### 3단계: 최소한의 작업 수행

프로그램을 사용할 수 있는지 확인하려면 다음 두 가지 간단한 경로 중 하나를 선택하세요.

#### 경로 A: 사진 열기

1. 파일 메뉴에서 로컬 이미지를 엽니다.
2. 이미지가 작업 공간에 표시될 수 있는지 확인
3. 속성 패널이나 관련 도구가 정상적으로 반응하는지 관찰합니다.

#### 경로 B: 시뮬레이션된 장치 또는 기존 장치 추가

1. 기기 관련 메뉴 진입
2. 시뮬레이션된 장치를 추가하거나 기존 장치를 연결합니다.
3. 해당 장치가 장치 영역에 나타나고 정상적으로 응답하는지 확인합니다.

기기 관련 콘텐츠는 [기기 서비스 개요](../01-user-guide/devices/overview.md)를 계속해서 확인하실 수 있습니다.

### 4단계: 실제 비즈니스에 더 가까운 항목을 시도해 보세요.

목표에 따라 후속 독서를 선택하십시오.

- 이미지를 보고 주석을 달고 싶다면: [이미지 편집기 개요](../01-user-guide/image-editor/overview.md)로 이동하세요.
- 장치 및 수집 링크를 구성하려는 경우: [장치 서비스 개요](../01-user-guide/devices/overview.md)로 이동합니다.
- 프로세스가 어떻게 설계되고 실행되는지 이해하려면 [워크플로 개요](../01-user-guide/workflow/README.md)로 이동하세요.

## 이 페이지의 경계를 한 문장으로 이해하세요

이 페이지는 사용자를 시스템에 연결하는 역할만 담당합니다. 설치 프로그램 구현, 모듈 설계 기록 또는 모든 클래스 라이브러리 세부 정보를 설명할 책임이 없습니다.
