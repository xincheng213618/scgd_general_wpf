# 첫 실행 가이드

이 안내서는 ColorVision을 처음 설치한 후 프로그램을 시작하고 기본 구성을 수행하는 방법을 배우는 데 도움이 됩니다.

## 프로그램 시작

### 방법 1: 시작 메뉴에서 시작

1. Windows 시작 메뉴를 엽니다.
2. "ColorVision" 또는 "ColorVision"을 찾으세요.
3. 아이콘을 클릭하여 프로그램을 시작하세요

### 방법 2: 바탕화면 바로가기에서 시작

설치 중에 바탕화면 바로가기를 생성하도록 선택한 경우 바탕화면에 있는 ColorVision 아이콘을 두 번 클릭하여 시작하십시오.

### 방법 3: 명령줄에서 시작(개발자)

``파워셸
cd "설치 디렉터리"
.\ColorVision.exe
```
## 처음 시작 시 자동 구성

프로그램은 처음 시작될 때 자동으로 다음 작업을 수행합니다.

1. **구성 파일 생성**
   - 위치: `%USERPROFILE%\Documents\ColorVision\`
   - 포함: 사용자 구성, 장치 구성, 프로세스 구성 등

2. **로그 시스템 초기화**
   - 로그 디렉터리: `%USERPROFILE%\Documents\ColorVision\logs\`
   - 로그 수준: 기본값은 정보입니다.

3. **스캔 플러그인**
   - `Plugins` 디렉토리를 자동으로 스캔합니다.
   - 사용 가능한 모든 플러그인을 로드합니다.

4. **메인 창 표시**
   - 메인 인터페이스 표시
   - 환영 메시지 표시(선택 사항)

## 기본 인터페이스 탐색

### 메인창 구성

```
┌──────────────────────────────────────────┐
│ 메뉴바 │
├──────────────────────────────────────────┤
│ 툴바 │
├──────────┬─────────────────────┬────────────┤
│ │ │ │
│ 장치 목록 │ 이미지 표시 영역 │ 속성 패널 │
│ │ │ │
│ │ │ │
├──────────┴─────────────────────┴─────────────┤
│ 상태 표시줄 │
└──────────────────────────────────────────┘
```
### 주요 영역 설명

- **메뉴 바**: 파일, 편집, 보기, 장치, 프로세스, 플러그인, 도움말 및 기타 메뉴
- **도구 모음**: 자주 사용하는 기능에 대한 단축키 버튼
- **장치 목록**: 연결되어 있고 사용 가능한 장치를 표시합니다.
- **이미지 표시 영역**: 이미지 표시, 프로세스 편집기 등
- **속성 패널**: 선택한 개체의 속성을 표시합니다.
- **상태 표시줄**: 프로그램 상태 및 프롬프트 정보를 표시합니다.

자세한 인터페이스 설명은 [메인 윈도우 가이드](../01-user-guide/interface/main-window.md)를 참조하세요.

## 기본 작동 데모

### 1. 이미지를 엽니다

**단계**:
1. 메뉴바 **파일** → **이미지 열기**를 클릭합니다.
2. 또는 단축키 'Ctrl+O'를 사용하세요.
3. 이미지 파일을 선택하세요(BMP, JPG, PNG, TIFF 및 기타 형식 지원)
4. '열기'를 클릭하세요.

이미지가 중앙 디스플레이 영역에 표시됩니다.

### 2. 시뮬레이션 장치 추가

**단계**:
1. 메뉴바 **장치** → **장치 추가**를 클릭합니다.
2. **아날로그 카메라** 또는 기타 아날로그 장치를 선택합니다.
3. 장치 매개변수 구성
4. "확인"을 클릭하세요

장치가 왼쪽 장치 목록에 나타납니다.

### 3. 로그 보기

**단계**:
1. 메뉴바 **도움말** → **로그**를 클릭합니다.
2. 또는 단축키 'Ctrl+L'을 사용하세요.
3. 로그창에서 실행 중인 프로그램 로그 보기

### 4. 로드된 플러그인 보기

**단계**:
1. 메뉴바 **플러그인** → **플러그인 관리**를 클릭합니다.
2. 로드된 플러그인 목록 보기
3. 플러그인 활성화/비활성화 가능

## 일반적인 첫 실행 문제

### 문제 1: 프로그램이 느리게 시작됩니다.

**이유**: 처음 시작하려면 초기화 구성, 플러그인 검색 등이 필요합니다.

**해결책**:
- 일반적으로 첫 번째 부팅에는 10~30초 정도 소요됩니다.
- 1분 이상 소요되는 경우 로그를 확인하여 구체적인 이유를 확인하세요.
- 플러그인 로딩 문제일 수 있습니다. 플러그인 디렉토리에서 플러그인을 일시적으로 제거해 보세요.

### 문제 2: 비정상적인 인터페이스 표시

**원인**: 해상도 또는 DPI 설정 불일치

**해결책**:
- 1920x1080 이상의 해상도를 권장합니다.
- DPI 스케일링을 100%로 설정
- Windows 설정에서 디스플레이 배율 조정

### 문제 3: 구성 파일을 찾을 수 없습니다.

**원인**: 권한 문제 또는 경로 문제

**해결책**:
- 사용자 문서 디렉터리에 액세스할 수 있는지 확인하세요.
- 관리자 권한으로 프로그램 실행
- 로그에서 오류 메시지를 확인하세요.

### 문제 4: 플러그인 로딩 실패

**원인**: 플러그인 종속성이 없거나 버전이 일치하지 않습니다.

**해결책**:
- 플러그인 로딩 오류 로그 확인
- 플러그인의 Manifest.json 파일을 확인하세요.
- 플러그인이 ColorVision 버전과 일치하는지 확인하세요.

더 궁금한 사항은 [문제 해결](../01-user-guide/troubleshooting/common-issues.md)을 참조하세요.
