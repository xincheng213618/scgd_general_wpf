# 시스템 아키텍처 개요(구 입구)

이 페이지는 이전 링크와의 호환성을 위해 유지됩니다. 현재 문서 구조는 "시작하기"와 "아키텍처 디자인"으로 분리되어 있으며, 시작하기 장에서는 시스템 아키텍처에 대해 더 이상 설명하지 않습니다.

1. [건축설계 장 개요](../../03-architecture/README.md)
2. [시스템 아키텍처 개요](../../03-architecture/overview/system-overview.md)

## 여기서는 더 이상 확장되지 않는 이유는 무엇인가요?

- 시작하기 장은 이제 설치, 첫 실행, 빠른 시작만 담당합니다.
- 디자인 경계, 모듈 관계, 런타임 구조가 [디자인 및 아키텍처](../../03-architecture/README.md)로 통합되었습니다.
- 이렇게 하면 "설치 요구 사항", "사용 지침" 및 "설계 분석"이 동일한 읽기 경로에 혼합되는 것을 방지할 수 있습니다.
