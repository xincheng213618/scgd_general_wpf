# 카메라 매개변수 구성

이 페이지에서는 "카메라 서비스 구성 페이지"에서 일반적으로 변경해야 할 사항에 대해 설명합니다. 여기서 조정되는 것은 논리적 카메라 서비스 매개변수이며, 기본값 중 일부는 바인딩된 물리적 카메라에서 나옵니다.

## 공통 구성 내용

### 바인딩 정보

- 현재 서비스 인스턴스 이름
-바인딩된 물리적 카메라 또는 CameraCode
- 현재 프로젝트에서 실제로 호출되는 카메라 서비스는 무엇인가요?

### 컬렉션 매개변수

- 채널 및 비트 깊이
- 촬영 모드
- 노출, 게인, 채도
- 자동 노출 스위치 및 관련 매개 변수

### 확장 매개변수

- 영상 관련 구성
- 파일 캐싱 및 저장 형식
- 모터, CFW, 자동 초점 및 기타 슬레이브 구성

## 순서를 조정하는 것이 좋습니다

1. 먼저 올바른 실제 카메라를 바인딩합니다.
2. 서비스가 일련의 기본 매개변수를 동기화하도록 합니다.
3. 그런 다음 노출, 게인, 채널 및 모드를 조정합니다.
4. 저장 후 실시간 이미지나 프로세스를 이용하여 검증을 진행합니다.
5. 고급 조정이 필요한 경우 템플릿 또는 교정 페이지로 들어갑니다.

## 서로 영향을 미칠 가능성이 가장 높은 매개변수는 무엇입니까?

- 카메라 모드 및 채널
- 비트 심도 및 획득 모드
- 노출 시간 및 게인
- 자동 노출 및 수동 노출 매개변수
- 물리적 카메라 기본값 및 현재 서비스 디스플레이 구성

특정 수정 후 그림이 확실히 비정상적인 경우 단일 매개변수에만 초점을 맞추는 대신 이러한 그룹이 함께 변경되었는지 확인하는 데 우선순위를 두십시오.

## FAQ

### 매개변수를 변경해도 효과가 없습니다.

- 변경하려는 카메라 서비스가 현재 사용 중인 카메라 서비스인지 확인하세요.
- 저장 후 뷰를 다시 열거나 서비스를 다시 시작하세요.
- 물리적 카메라 동기화 구성에 의해 재정의되었는지 확인

### 무리한 노출이나 채널 설정

- 먼저 현재 카메라 모드에서 어떤 채널을 지원하는지 확인하세요.
- 3채널 매개변수와 단일 채널 매개변수를 혼합하지 마십시오.
- 노출 과다나 노이즈가 발생하는 경우 노출 및 게인 조정을 우선적으로 고려하세요.

### 저장 후에도 처리결과가 여전히 비정상입니다.

- 실시간 화면을 이용하여 영상 획득이 정상적인지 먼저 확인합니다.
- 진행 중인 템플릿 매개변수가 여전히 이전 값인지 다시 확인하세요.
- 필요한 경우 보정 및 자동 노출 템플릿 확인 연결
