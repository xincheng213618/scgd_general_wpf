# 장치 추가 및 구성

이 페이지에서는 장치 서비스를 추가하는 방법, 저장하는 방법, 매개변수를 변경할 때 먼저 확인할 위치 등 사용자 작업 경로에 대해서만 설명합니다.

## 일반적인 시나리오

- 새 장치를 연결하려면 먼저 시스템에 해당 서비스를 생성해야 합니다.
- 장치가 이미 존재하지만 이름, 통신 매개변수 또는 바인딩 개체를 수정해야 합니다.
- 프로젝트 전환 후 장비 순서 조정, 기존 장비 비활성화, 신규 장비 추가 등의 작업이 필요합니다.

## 기본과정

1. 장치 관리 창을 엽니다.
2. 장치 설정 또는 편집 인터페이스로 들어갑니다.
3. 해당 유형의 새로운 장치 서비스를 추가하거나 기존 장치를 선택하여 수정하세요.
4. 필요한 매개변수를 입력하고 저장합니다.
5. 장치 목록으로 돌아가서 해당 장치가 사용 가능한 목록에 나타나는지 확인합니다.

후속 프로세스, 카메라 창 또는 결과 인터페이스에서 장치가 여전히 표시되지 않으면 먼저 장치 목록으로 돌아가 이 단계가 실제로 적용되는지 확인하십시오.

## 일반적으로 구성 시 확인해야 하는 내용

### 기본정보

- 기기 이름은 쉽게 구별할 수 있나요?
- 장치 유형이 올바르게 선택되어 있습니까?
- 현재 프로젝트가 올바른 장치 인스턴스를 참조하는지 여부

### 통신 또는 바인딩 정보

- 네트워크 주소, 포트, 주체 또는 통신 식별자가 올바른지 여부
- 해당 물리적 장치 또는 리소스 개체가 바인딩되어 있는지 여부
- 기본 드라이버, 서버 또는 타사 프로그램이 준비되어 있는지 여부

### 저장 후 표시 여부

- 해당 장치가 이미 장치 목록에 표시되어 있는지 여부
- 기기 상태는 정상인가요?
- 해당 창이나 프로세스 노드에서 장치를 선택할 수 있는지 여부

## FAQ

### 추가 후 목록에 장치가 없습니다

- 실제로 저장이 성공했는지 확인하세요.
- 잘못된 프로젝트나 잘못된 구성 컨텍스트에서 작업하고 있는지 확인하세요.
- 장치 창을 다시 열어 목록만 새로 고쳐지지 않는지 확인하세요.

### 매개변수가 변경되었지만 장치 동작은 변경되지 않았습니다.

- 다시 연결해야 하는지, 컨트롤을 다시 생성해야 하는지, 해당 창을 다시 열어야 하는지 확인하세요.
- 현재 사용 중인 디바이스 인스턴스에 대한 수정인지 확인

### 구성 페이지에 무엇을 입력해야 하는지 모르겠습니다.

- 먼저 해당 장비 특별 페이지로 이동하여 매개변수 설명을 확인하세요.
- 장치가 물리적 장치 개체에 의존하는 경우 먼저 물리적 장치 관리 또는 교정 단계를 완료하세요.
