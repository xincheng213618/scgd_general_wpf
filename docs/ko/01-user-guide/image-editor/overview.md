# 이미지 편집기 개요

이미지 편집기는 ColorVision에서 결과를 보고 상호 작용하는 데 가장 일반적으로 사용되는 영역 중 하나입니다. 여기에서는 이미지 보기, 확대/축소 및 이동, ROI 그리기, 주석 만들기, 비디오 재생, 특정 시나리오에 대한 3D 또는 의사 색상 보기로 전환 등을 수행할 수 있습니다.

## 여기서 무엇을 할 수 있나요?

- 일반적인 이미지 결과 보기
- 화면 확대/축소, 이동 및 맞춤
- 직사각형, 원, 다각형, 곡선 및 텍스트 주석 그리기
- 속성 패널을 보고 선택한 개체를 조정합니다.
- 가색상, 히스토그램 또는 3D 보기를 사용하여 데이터 관찰
- 비디오 파일 장면에서 직접 재생, 일시 정지, 드래그 진행 및 미리 보기 확대/축소 전환

## 일반적인 사용 순서

1. 이미지나 결과 창을 열어 시작하세요.
2. 확대/축소 및 이동을 사용하여 주요 영역을 식별합니다.
3. 측정하거나 표시해야 할 경우 그리기 도구로 전환하세요.
4. 개체 매개변수를 확인해야 할 경우 속성 패널을 엽니다.
5. 입력이 비디오인 경우, 추가 분석 전에 먼저 재생이 원활한지 확인하십시오.

## 비디오 모드에서는 어떻게 되나요?

지원되는 비디오 파일이 열리면 이미지 편집기가 비디오 작업 모드로 전환됩니다. 이 시점에서 일반적으로 다음을 볼 수 있습니다.

- 재생 및 일시 정지 제어
- 진행률 표시줄
- 이중 속도 전환
- 미리보기 확대/축소 토글
- 도구 모음 관련 옵션을 음소거하거나 자동으로 숨깁니다.

고해상도 비디오인 경우 처음부터 기본 해상도로 이동하는 것보다 먼저 미리보기 확대/축소를 낮추는 것이 더 실용적인 경우가 많습니다.

## 특히 어떤 시나리오가 일반적인가요?

### 테스트 결과 보기

- 먼저 원본 이미지와 결과 Coverage가 일치하는지 확인합니다.
- ROI, 주석, 속성이 올바른지 다시 확인하세요.

### 이미지 표시 조정

- 확대/축소, 가색상 및 대비 관찰의 사용을 우선시합니다.
- 처음에는 기본 구성을 수정하지 말고 먼저 표시 문제가 있는지 확인하십시오.

### 동영상 또는 동적 데이터 확인

- 먼저 재생이 정상적인지 확인하세요.
- 미리보기 비율을 줄일지, 일부 추가 디스플레이를 끌지 결정하세요.

## FAQ

### 영상은 정상적으로 나오나 동작이 느린 느낌

- 뷰를 축소하거나 동시에 표시되는 내용을 줄이는 것부터 시작하세요.
- 비디오 장면에서는 먼저 미리보기 확대/축소를 줄입니다.
- 3D 보기 시나리오에서는 먼저 대상 해상도를 낮추거나 일반 보기로 다시 전환하여 문제의 범위를 확인합니다.

### 라벨이나 ROI를 선택하기가 쉽지 않습니다.

- 먼저 아직 그리기 모드에 있는지 확인하세요.
- 개체를 클릭하기 전의 선택된 상태로 돌아가 보세요.
- 필요한 경우 [속성 편집기](../interface/property-editor.md)를 사용하여 현재 선택된 개체를 확인합니다.

### 영상의 싱크가 맞지 않거나 매끄럽지 않습니다.

- 일시 중지했다가 다시 시작
- 다시 점프하려면 진행률 표시줄을 드래그해 보세요.
- 확대/축소 후 검토를 위해 저해상도 미리보기 비디오 우선순위 지정

### 3D뷰 실행 후 효과가 비정상적입니다.

- 먼저 입력 데이터 자체가 높이나 강도 시각화에 적합한지 확인합니다.
- 대상 해상도를 낮추거나 색상 매핑을 전환해야 하는지 다시 확인하세요.
