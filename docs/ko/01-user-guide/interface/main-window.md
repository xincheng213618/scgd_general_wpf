# 메인 창 탐색

메인 창은 ColorVision의 작업대입니다. 장비, 프로세스, 이미지 보기, 로그, 검색, 상태 모니터링 등 대부분의 일상 작업이 여기에 입력됩니다.

## 메인창에 진입 후 가장 먼저 살펴봐야 할 곳

### 상단 메뉴 및 검색 영역

- 메뉴는 주요 기능 입구입니다.
- 명령, 페이지 또는 개체를 빠르게 찾을 수 있는 검색 상자
- 어디에 기능이 숨겨져 있는지 모르신다면 먼저 검색을 이용해 보세요

### 왼쪽 탐색 영역

- 일반적으로 프로젝트, 리소스 트리 또는 관련 보기 탭을 보는 데 사용됩니다.
- 현재 페이지에 내용이 많은 경우 일반적으로 왼쪽 구조부터 시작하는 것이 더 빠릅니다.

### 중간 메인 작업공간

- 현재 운용중인 핵심컨텐츠 표시
- 이미지 보기, 프로세스 창, 템플릿 창 또는 기타 비즈니스 인터페이스일 수 있습니다.
- 현재 작업 모드에 따라 레이아웃이 단일 보기 또는 다중 보기로 전환될 수 있습니다.

### 하단 상태 표시줄

- 시스템 상태, 프롬프트 정보 및 일부 바로가기 항목을 관찰하는 데 사용됩니다.
- 장치, 서비스 또는 플러그인 상태가 비정상적인 경우 먼저 여기에 메시지가 있는지 확인하십시오.

## 메인창에서는 주로 무엇을 하시나요?

- 장치 관련 창 열기
- 프로세스 설계 및 실행 페이지 진입
- 이미지 또는 결과 창 보기
- 개방형 로그, 터미널 및 구성 인터페이스
- 검색 기능 입구 또는 물체 위치

## 권장되는 숙지 순서

1. 먼저 메인 메뉴, 검색 영역, 상태 표시줄에 대해 알아보세요.
2. 왼쪽 탐색 기능과 기본 작업 공간이 어떻게 협력하는지 이해합니다.
3. 그런 다음 [속성 편집기](./property-editor.md) 및 [이미지 편집기 개요](../image-editor/overview.md)를 입력합니다.
4. 하드웨어에 관해서는 [장치 서비스 개요](../devices/overview.md)를 계속해서 살펴보세요.
5. 자동화가 필요한 경우 [Workflow](../workflow/README.md)를 입력합니다.

## FAQ

### 특정 기능 입구를 찾을 수 없습니다

- 먼저 검색창을 이용해 키워드를 검색하세요.
- 해당 챕터 홈페이지로 돌아가서 재접속
- 플러그인 기능인 경우 플러그인이 로드되었는지 확인하세요.

### 창문이 너무 많이 열려 있는데 뭘 보고 있는지 모르겠어요.

- 현재 개체를 다시 확인하려면 왼쪽 탐색으로 돌아갑니다.
- 메인 작업공간 제목과 상태 표시줄 정보를 관찰하세요.
- 무작정 여러 창을 전환하지 않고 업무에 따라 해당 페이지를 다시 입력하세요.

### 예상한 메시지가 상태 표시줄에 나타나지 않습니다.

- 먼저 해당 서비스, 장치 또는 플러그인이 실제로 시작되었는지 확인하십시오.
- 자세한 내용은 [로그 뷰어](./log-viewer.md)를 다시 확인하세요.
