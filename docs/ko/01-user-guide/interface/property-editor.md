#속성 편집기

ColorVision은 메타데이터 기반 속성 편집기를 사용하여 객체 속성을 균일하게 표시하고 편집합니다. 대부분의 템플릿, 장치 및 구성 개체는 동일한 PropertyGrid 메커니즘을 통해 매개 변수를 노출합니다.

## 무엇을 보게 될까요?

- 왼쪽 또는 팝업창에 속성 목록 표시
- 카테고리별로 그룹화된 매개변수 영역
- 텍스트, 열거형, 부울, 색상, 파일 경로 등 다양한 유형에 대해 자동으로 전환되는 편집 컨트롤

## 일반적인 사용법

1. 개체 트리, 템플릿 편집기 또는 장치 구성 인터페이스에서 개체를 선택합니다.
2. 속성 패널에서 그룹화된 매개변수를 확인하세요.
3. 값을 수정한 후 인터페이스 또는 개체 상태가 즉시 업데이트되는지 관찰합니다.
4. 재설정 또는 확인 버튼이 있는 경우 기본 구성 파일을 직접 수동으로 변경하는 대신 버튼을 사용하는 데 우선 순위를 두십시오.

## 이 편집기는 무엇에 의존하나요?

- 속성에 대한 `Category`, `DisplayName`, `Description`과 같은 메타데이터
- 필요한 경우 사용자 정의 편집기 유형을 사용하여 제어 스타일을 결정합니다.
- 일부 복잡한 개체는 편집기에서 간단한 문자열 대신 중첩 구조로 확장됩니다.

- 이 페이지에서는 Perspective 사용에 대해서만 설명하고 더 이상 PropertyGrid의 구현 세부 사항을 다루지 않습니다.
- 속성 편집기 동작을 수정하려면 `UI/ColorVision.UI/` 및 `Engine/ColorVision.Engine/Templates/`의 기존 메타데이터 주석에서 코드를 살펴보는 것이 좋습니다.
