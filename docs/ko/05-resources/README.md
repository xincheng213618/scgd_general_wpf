#부록 및 자료

이 부분은 색인이나 부록으로 적합한 장기적으로 안정적인 콘텐츠만 보유하고 있으며 더 이상 흩어져 있는 주제 설명을 전달하지 않습니다.

## 현재 예약된 콘텐츠

- [프로젝트 구조 개요](./project-structure/README.md)
- [모듈 및 문서 비교표](./project-structure/module-documentation-map.md)
- [소프트웨어 라이센스 계약](./legal/software-agreement.md)
- [CHANGELOG.md](https://github.com/xincheng213618/scgd_general_wpf/blob/master/CHANGELOG.md)

## 사용법

- 창고 구조를 이해하고 싶다면 먼저 [프로젝트 구조 개요](./project-structure/README.md)를 읽어보세요.
- 모듈에 해당하는 문서 항목을 빠르게 찾고 싶은 경우: [모듈 및 문서 비교표](./project-structure/module-documentation-map.md)를 참조하세요.
- 버전 변경을 확인하고 싶은 경우 : 웨어하우스 루트 디렉터리의 'CHANGELOG.md'를 직접 확인하세요.
- 법률 텍스트를 보려면 [소프트웨어 라이센스 계약](./legal/software-agreement.md)을 입력하세요.

## 이 합의의 경계

- 이전 API v1.1, 템플릿, 데이터 저장소 및 기타 항목은 더 이상 리소스 영역 탐색에 배치되지 않습니다.
- 주제별 콘텐츠는 배포 가이드로 돌아가고, 인터페이스는 API 참조로, 시스템 디자인은 아키텍처 디자인으로 돌아가는 등 해당 장으로 돌아가야 합니다.
