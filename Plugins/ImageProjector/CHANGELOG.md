# Changelog

## [1.1.0.0] - 2024

### Added
- Multi-image list management (add/remove/move up/move down)
- Previous/Next buttons to switch images during projection
- Image stretch mode selection (Uniform/Fill/None/UniformToFill)
- Configuration persistence (image list, selected index, monitor, stretch mode)
- Localization for new features in all 7 languages

### Changed
- UI redesigned with two-column layout
- Enhanced monitor selection with stretch mode option

## [1.0.0.1] - 2024

### Added
- Initial release
- Multi-monitor support with visual layout
- Image projection to selected monitor
- Support for TIF, PNG, JPG, BMP formats
- ESC key to close projection
- Localization support (Chinese/English)
