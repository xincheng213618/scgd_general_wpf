# ImageProjector

图片投影工具 - 将图片全屏投影到指定显示器上。

## 功能特点

- 支持多显示器选择，可视化显示器布局
- 支持常见图片格式 (TIF, PNG, JPG, BMP)
- **多图片管理** - 添加、删除、上移、下移图片列表
- **投影中切换** - 上一张/下一张按钮快速切换投影图片
- **显示模式** - 适应、拉伸、居中、填充四种显示模式
- **配置持久化** - 自动保存图片列表、选中项、显示器和显示模式
- 全屏投影到选定显示器
- ESC 键快速关闭投影

## 使用方法

1. 点击"添加"按钮添加图片到列表（支持多选）
2. 选择目标显示器
3. 选择显示模式（适应/拉伸/居中/填充）
4. 点击"投影"按钮开始投影
5. 投影中可使用"上一张"/"下一张"按钮切换图片
6. 按 ESC 键或点击"停止投影"按钮关闭

## 显示模式说明

| 模式 | 说明 |
|------|------|
| 适应 | 保持宽高比，完整显示图片（默认） |
| 拉伸 | 拉伸填满整个屏幕，可能变形 |
| 居中 | 原始尺寸居中显示，不缩放 |
| 填充 | 保持宽高比填满屏幕，可能裁剪 |

## 依赖

本插件仅依赖 ColorVision.UI，体积小巧。
