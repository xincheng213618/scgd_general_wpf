# Conoscope 优化建议计划 2026-05-20

来源: `优化建议_20260520.docx` 中非灰色或未完全灰色的条目。

## 本轮已处理

1. 观察相机设备列表默认仅显示型号中包含 `CS200` 的设备，并在观察相机设置中提供“仅显示 CS200 设备”配置项，默认启用。
2. 观察相机列表中的 CS200 型号显示简化为 `MV-CS200`，保留序列号用于区分设备。
3. 图像工具栏“参考拖拽”改名为“截面标记”。
4. 预处理页的手动应用按钮改名为“重新应用”。
5. 图像上的伪彩色带改为横向分布。
6. 删除图像上方工具栏中与主功能区重复的 `3D` 按钮。
7. 关注点橡皮擦限制为只删除关注点圆，避免误删测试区域、截面线等其他框线。
8. 色域/对比度记录按钮在未记录前显示“记录 X”，记录后显示“已记录 X”，并修正记录详情提示。
9. 主功能区已保留“计算对比度”按钮，本轮同步检查其入口状态。

## 后续计划

1. 主界面图像区域与曲线区域布局优化，目标比例按“图像:曲线 = 1:2”评估。
2. 增加实时关注点功能，在图上显示实时关注点取值范围圆圈。
3. 关注点编辑增强: 坐标和半径支持手动输入，显示单位从像素转换为角度，并支持保存关注点模板。
4. RGB 三张图复用同一套关注点，避免导入 G/B 后重复绘制。
5. 对比度结果继续完善: 核对结果弹窗字段、角度坐标、亮/暗亮度、对比度表格，并评估是否需要在显示通道中增加对比度结果显示。
6. 主界面图标和执行按钮视觉统一放大、增强主操作突出度，等主要功能稳定后整体调整。
7. 伪彩色显示范围继续跟踪客户反馈；当前渲染已按最小值到最大值归一化，如仍不明显再增加手动范围或百分位范围配置。
