# ColorVision Plugins

## 插件系统概述

ColorVision 插件目录，包含系统扩展插件和第三方功能模块。

## 标准插件列表

- **EventVWR** - Windows事件查看器集成
- **SystemMonitor** - 系统性能监控工具  
- **Pattern** - 图案分析和生成插件
- **WindowsServicePlugin** - Windows服务管理插件
