# CHANGELOG

## [Unreleased]

1. 增强Reset功能：添加用户默认配置层
   - 新增"保存默认"按钮，可将当前配置保存为用户默认值
   - Reset按钮现在先重置到用户默认值（如果存在），否则重置到类默认值
   - 属性编辑器添加"恢复出厂"按钮，可直接重置到类默认值
   - 用户默认值存储在 Documents/ColorVision/Pattern/UserDefaults/ 目录
