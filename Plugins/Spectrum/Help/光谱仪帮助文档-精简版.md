# 光谱仪帮助文档（精简版）

适用对象：终端用户、现场工程师  
阅读时长：5-10 分钟

## 1. 快速上手

1. 连接光谱仪并确认设备在线。
2. 先做一次校零（暗电流校正）。
3. 选择测量模式（普通模式 / 光通量模式）。
4. 开始测量并查看结果。
5. 需要留档时导出 CSV。

## 2. 最常用功能

### 自动校零
- 作用：去除暗电流和背景噪声。
- 什么时候做：
  - 开机预热后
  - 积分时间变化后
  - 环境温度明显变化后

### 自动积分时间
- 作用：自动找到合适积分时间，避免过暗或过曝。
- 建议：和自适应校零一起使用，结果更稳定。

### 连续测量
- 作用：批量采集数据，用于稳定性评估。
- 注意：连续测量中部分按钮会被锁定，结束后恢复。

### 光通量模式（EQE）
- 作用：同时得到 EQE、光通量、辐射通量、光效。
- 必填：电压(V)、电流(mA)（手动输入或源表读取）。

## 3. 常见指标速查

- Ra：显色性指数，越高颜色还原越好。
- CCT：相关色温，反映冷暖感。
- Lp：峰值波长，光谱峰值位置。
- Ld：主波长，感知颜色对应波长。
- FWHM：半波宽，越小通常颜色越纯。
- EQE：外量子效率，电子转光子效率。
- 光通量（lm）：视觉加权总光输出。
- 辐射通量（W）：总辐射功率。
- 光效（lm/W）：单位功率产生的可见光。
- Duv：偏离黑体轨迹程度，越接近 0 越接近标准白光轨迹。

## 4. 推荐参数与经验

- IP（积分峰值比）建议控制在 30%-80%。
- IP < 10%：信号偏弱，可增大积分时间。
- IP > 90%：有饱和风险，可减小积分时间。
- PWM 光源建议开启同步频率测量。

## 5. 导出与复算

- 导出：在数据列表选择条目后导出 CSV。
- 复算：在 EQE 模式下可修改 V/I 后重新计算。

## 6. 常见问题

### 测量值跳动大
- 检查是否需要开启同步频率（PWM 光源）。
- 重新执行校零。
- 确认积分时间是否过短。

### 结果偏暗或噪声大
- 提高积分时间或启用自动积分。
- 检查入光口是否被遮挡。

### 出现饱和
- 降低积分时间。
- 减小光强或调整距离。

## 7. 建议阅读顺序

1. 自动校零
2. 自动积分时间
3. 光通量模式（如需测 EQE）
4. 数据导出
