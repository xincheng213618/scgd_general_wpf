# Treemap Plugin

A high-performance WPF Treemap control similar to tools like SpaceSniffer.

## Features
- Squarified treemap layout for optimal aspect ratios
- DrawingVisual-based rendering for thousands of rectangles
- Hover tooltips showing node name and size
- Small node filtering (nodes < 2x2 pixels are skipped)
- Mock file system demo

## Usage
Open from the Tools menu → Treemap Viewer.
