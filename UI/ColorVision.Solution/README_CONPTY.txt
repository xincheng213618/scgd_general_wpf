🔧 使用方法
--------------------------------------------------------------------------------

通过 UI: 
  应用程序菜单 → 帮助 → 终端

编程方式:
  var terminal = new ConPtyTerminal();
  terminal.OutputReceived += (s, output) => Console.Write(output);
  terminal.Start(80, 25, "cmd.exe");
  terminal.SendInput("dir\r\n");
  terminal.Dispose();
