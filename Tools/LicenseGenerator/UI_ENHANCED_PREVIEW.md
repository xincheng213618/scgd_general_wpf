## 界面元素详解

### 输入字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 机器码 | 文本框 + 按钮 | ✅ | 十六进制字符串，可使用"使用当前"按钮自动填充 |
| 客户名称 | 文本框 | ✅ | 被许可人或客户公司名称 |
| 设备型号 | 文本框 | ❌ | 设备型号标识，如 "CV-CAM-1000" |
| 有效期至 | 日期选择器 | ✅ | 许可证过期日期，默认为 1 年后 |
| 剩余天数 | 只读标签 | - | 自动计算并显示，颜色提示：绿(>30天)/橙(≤30天)/红(已过期) |
| 签发机构 | 文本框 | ❌ | 默认 "ColorVision" |

### 按钮功能

- **使用当前**：将当前机器码填入输入框
- **复制** (机器码)：复制当前机器码到剪贴板
- **生成许可证**：生成增强格式的许可证（Base64 + JSON）
- **复制** (许可证)：复制生成的许可证到剪贴板
- **保存**：保存许可证到 .lic 文件

### 状态信息

显示当前操作状态，颜色编码：
- **绿色**：操作成功
  - "就绪"
  - "许可证生成成功 (验证: 通过 ✓, 有效期: 365 天)"
  - "许可证已复制到剪贴板"
- **红色**：错误或失败
  - "请输入机器码"
  - "请输入客户名称"
  - "有效期必须晚于当前时间"

### 剩余天数颜色提示

```
剩余天数 > 30:   绿色 ✓  (正常)
剩余天数 ≤ 30:   橙色 ⚠  (即将过期)
剩余天数 ≤ 0:    红色 ✗  (已过期)
```

## 交互流程

### 生成增强许可证的完整流程

### 操作流程

```
1. 打开工具
   ↓
2. 填写必填字段
   - 机器码 (可点击"使用当前")
   - 客户名称
   - 有效期 (默认1年后)
   ↓
3. (可选) 填写选填字段
   - 设备型号
   - 签发机构 (默认已填)
   ↓
4. 观察剩余天数显示
   - 实时更新
   - 颜色提示
   ↓
5. 点击"生成许可证"
   ↓
6. 查看输出
   - Base64 编码的许可证字符串
   - 状态栏显示验证结果和有效期
   ↓
7. 复制或保存
   - 点击"复制"按钮 → 剪贴板
   - 点击"保存"按钮 → .lic 文件
```

### 输入验证

工具会自动验证：
1. **机器码不能为空**
2. **客户名称不能为空**
3. **必须选择有效期**
4. **有效期必须晚于当前时间**

验证失败时：
- 状态栏显示红色错误消息
- 焦点自动移到出错的输入框

### 自动功能

1. **自动清空输出**
   - 任何输入字段变化时，自动清空许可证输出框
   - 防止使用错误的旧许可证

2. **自动验证**
   - 生成许可证后立即验证
   - 在状态栏显示验证结果

3. **剩余天数实时更新**
   - 修改有效期时自动更新剩余天数
   - 根据剩余天数自动调整颜色

## 版本说明

### 当前版本
- 专注于增强型许可证生成
- 简化的单页面界面
- 更清晰的工作流程
- 完整的许可证信息管理

### 与旧版本对比

**旧版本**：

**新版本**（当前）：
- 移除简单许可证模式（已弃用）
- 专注于功能更强大的增强型许可证
- 单一界面，更简洁直观
- 简单许可证方法已标记为 `[Obsolete]`，仅用于向后兼容
- 窗口大小 750x600
- 可调整大小
- 剩余天数显示
- 完善的输入验证
- 详细的状态反馈
